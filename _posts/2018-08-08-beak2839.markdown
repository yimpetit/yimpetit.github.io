풀이과정
1. 주어진 설탕 n 킬로를 5킬로 자루에 최대한 담는다.
2. 남은 설탕들을 3킬로 자루에 모두 담을 수 없다면 5킬로 자루를 하나씩 해체하여 계산한다.
3. 계산된 자루의 갯수를 합한다.

{% highlight javascript %}
var fs = require('fs');
var n = fs.readFileSync('/dev/stdin').toString()*1;
var five = Math.floor(n/5);
n %= 5;
var three = 0;
while(five>=0){
  if(n%3==0){
    three = n/3;
    n = 0;
    break;
  }else{
    five --;
    n += 5;
  }
}
console.log(n==0?five + three : -1);
{% endhighlight %}
